Hi. It's simple json repository based on Flask. Enjoy it.

To find files by tag use URL: http://127.0.0.1:5000/storage/files/'tag'/
To update tag with existing file use URL: http://127.0.0.1:5000/storage/files/"tag"/"filename"/
Where "tag" - Tag you are searching for
"filename" - origin filename
